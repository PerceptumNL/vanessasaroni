## Brand Voice Rules
Voice: **Friendly, Genuine, Witty, Passionate**

### IS vs IS NOT
| IS | IS NOT |
|----|--------|
| Sophisticated | Ostentatious/Snob |
| Classic | Outdated |
| Up-to-date | Trend victim |
| Sustainable | Ecology cliche |
| Inspiring | Naive |
| Warm, personal, inviting | Cold, corporate, salesy |
| Story-driven, emotional | Feature-listing, specs-heavy |
| Culturally rich, worldly | Generic, any-brand |
| Passionate about craft | Preachy about sustainability |
| Intimate, exclusive | Mass-market, broadcast |

### Absolute Rules
- NEVER discount. NEVER negotiate on price. NEVER run a "sale"
- NEVER use "cheap", "affordable", "budget-friendly" — use "accessible" if needed
- NEVER use stock photography
- NEVER position sustainability as primary selling point (it's supporting, not headline)
- Every communication should feel like it comes from Vanessa herself, not a brand account
- Prefer educating about the product over generic text
- Talk about intangible benefits, not functional specs
- Use emotional approach easy to connect with
- Witty product descriptions (e.g. "Class used to be priceless. Now from €675")

## Website Strategy
- Shift from e-commerce to **luxury lookbook** with inquiry-based conversion
- Replace "Add to Cart" with "Inquire About This Piece" / "Book a Private Viewing"
- Each of 10 designs needs its own story page (artisan, fabric, inspiration)
- Scarcity indicators: stock counts per design
- Mobile-first (audience comes from Instagram)
- Keep direct purchase as secondary path
